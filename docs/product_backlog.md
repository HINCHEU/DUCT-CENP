# Product Backlog

The Product Backlog is a prioritized list of features, bug fixes, and technical tasks required to build the Duct-Cenp Order Management System.

| Priority | ID | Type | User Story / Feature | Estimation (Story Points) |
|----------|----|------|----------------------|---------------------------|
| 1 | PB-01 | Feature | **Authentication System:** As a user, I want to securely log in and out so that my data is protected. | 3 |
| 2 | PB-02 | Feature | **Role Management:** As an admin, I want to assign roles (Engineer, Manager, Workshop) to users so they have appropriate access rights. | 3 |
| 3 | PB-03 | Feature | **Database Schema:** As a developer, I need to design and migrate the database schema for Sites, Duct Types, Orders, and Items. | 5 |
| 4 | PB-04 | Feature | **Order Creation:** As an Engineer, I want to select a site and create a draft order. | 5 |
| 5 | PB-05 | Feature | **Order Items:** As an Engineer, I want to add multiple duct items with distinct sizes and quantities to my draft order. | 8 |
| 6 | PB-06 | Feature | **Order Submission:** As an Engineer, I want to submit a draft order so a manager can review it. | 2 |
| 7 | PB-07 | Feature | **Manager Review Dashboard:** As a Manager, I want to see a list of submitted orders needing my approval. | 3 |
| 8 | PB-08 | Feature | **Manager Editing & Approval:** As a Manager, I want to edit quantities and approve/reject orders to ensure accuracy. | 5 |
| 9 | PB-09 | Feature | **Workshop Dashboard:** As a Workshop user, I want to see a queue of approved orders ready to be manufactured. | 3 |
| 10 | PB-10 | Feature | **Status Updates:** As a Workshop user, I want to update the status of an order as I work on it. | 2 |
| 11 | PB-11 | Feature | **PDF Cut-List Report:** As a user, I want to download a formatted PDF of the order grouped by duct type for physical printing. | 8 |
| 12 | PB-12 | Feature | **Order Comments:** As a user, I want to leave comments on an order to communicate with other departments. | 5 |
| 13 | PB-13 | Enhancement | **Email Notifications:** As a user, I want to receive an email when an order's status changes. | 5 |
| 14 | PB-14 | Enhancement | **Inventory Linking:** As a Manager, I want to see if we have enough raw material in stock before approving. | 13 |
